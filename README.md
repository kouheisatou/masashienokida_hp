# Masashi Enokida Homepage

This project serves as the personal homepage for Masashi Enokida, featuring a modular design with various customizable themes to showcase different styles and content presentations. Built with React and Vite, it offers a fast and modern development experience.

## Features

-   **Multiple Themes:** Easily switch between different visual themes (Classic, Modern, Cinematic, Nature, Nocturne, Royal Classic, Editorial Classic, Stage, Studio) to match various moods or content types.
-   **Responsive Design:** Optimized for a seamless experience across all devices.
-   **Modular Component Architecture:** Designed with reusable React components for maintainability and scalability.

## Technologies Used

-   [React](https://react.dev/)
-   [TypeScript](https://www.typescriptlang.org/)
-   [Vite](https://vitejs.dev/)
-   [Lucide React](https://lucide.dev/icons/) for beautiful and customizable icons

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/masashienokida_hp.git
    cd masashienokida_hp
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Development Server

To start the development server:

```bash
npm run dev
```

This will typically open the application in your browser at `http://localhost:5173`.

### Building for Production

To build the application for production:

```bash
npm run build
```

The optimized production build will be generated in the `dist` directory.

### Previewing the Production Build

To preview the production build locally:

```bash
npm run preview
```
