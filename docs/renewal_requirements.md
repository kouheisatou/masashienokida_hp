# リニューアルサイト要件定義

現行サイトの分析および追加要望に基づき、リニューアル後のサイトに必要な機能とページ構成をまとめました。

## 1. サイト全体の基本要件
*   **デザイン**: `docs/design_guide.md` に基づく、「劇場のようなゴージャスで落ち着いた雰囲気」かつ「モダンで見やすい」デザイン。
*   **レスポンシブデザイン**: スマートフォン、タブレット、PCなど全デバイスで最適に表示されること。
*   **SEO最適化**: 検索エンジン（Google等）で適切にヒットするためのメタデータ設定、構造化データ。
*   **高速化**: 画像の最適化、コードの軽量化による読み込み速度の向上。
*   **認証機能**: Google Authを用いたログイン機能（会員・管理者）。
*   **決済機能**: Stripeを用いた会費決済・寄付機能。

## 2. システム構成・技術スタック
*   **構成**: モノレポ (Monorepo) 構成
    *   `frontend`: ユーザー向けサイト + 管理者画面 (Next.js推奨)
    *   `backend`: APIサーバー (Node.js/NestJS または Go 等)
*   **環境構築**: Docker Compose
    *   フロントエンド、バックエンド、DB等を `docker-compose up` 一発で起動。
    *   環境変数 (`.env`) はルートディレクトリに1つ配置し、Docker Compose経由で各コンテナに共有。
*   **インフラ**: (要検討) Vercel, AWS, GCP 等

## 3. ページ構成と必要機能

### A. 一般ユーザー向け (Public)

#### 1. HOME (トップページ)
*   **メインビジュアル**: アーティストの魅力を伝える高品質なヒーロー画像または動画。
*   **最新情報 (News/Topics)**: ブログやコンサート情報の最新記事を自動表示。
*   **ピックアップ**: 最新アルバムや直近の重要コンサートへのリンク。
*   **SNSリンク**: 各種SNSへの導線。

#### 2. BIOGRAPHY (プロフィール)
*   **プロフィール表示**: 日本語および英語での経歴表示。
*   **アーティスト写真**: 宣材写真のギャラリー的要素。

#### 3. CONCERT (コンサート情報)
*   **公演一覧表示**: 今後の公演スケジュールをリスト表示（日時、会場、プログラム）。
*   **アーカイブ機能**: 過去の公演情報を「HISTORY」として自動的にアーカイブ。
*   **チケット購入導線**: 外部サイトリンクまたは問い合わせ。

#### 4. HISTORY (活動履歴)
*   **年表形式の活動記録**: 過去のリサイタルや主要な活動を年度ごとに整理。
*   **レビュー/コメント掲載**: 公演に対する批評やコメント。

#### 5. DISCOGRAPHY (ディスコグラフィ)
*   **CD/アルバム一覧**: ジャケット画像、タイトル、収録曲、発売日。
*   **購入リンク**: 音楽センター等の販売サイトへの外部リンク。

#### 6. SUPPORTERS (ファンクラブ案内)
*   **会員制度案内**: 無料会員・有料会員（ゴールド会員）の違い。
*   **入会・決済**: Stripe決済によるオンライン入会・継続課金。

#### 7. BLOG (ブログ)
*   **記事一覧**: サムネイル、タイトル、日付、カテゴリ、抜粋。
*   **記事詳細**: テキスト、画像、動画の埋め込み。
*   **カテゴリ/タグ**: 「コンサート案内」「メディア出演」など。

#### 8. CONTACT (お問い合わせ)
*   **お問い合わせフォーム**: 演奏依頼、ファンクラブ関連、その他。

### B. 会員限定エリア (Members Only)

*   **ログイン/マイページ**: Google Authによるログイン。会員ステータス確認。
*   **限定コンテンツ**: 会員のみが閲覧できる記事、動画、先行予約情報。
*   **会員情報の管理**: 登録情報の確認・変更、退会処理。

### C. 管理者機能 (Admin Dashboard)

*   **ダッシュボード**: サイトのアクセス概況や会員数などのサマリー。
*   **コンテンツ管理 (CMS機能)**:
    *   **お知らせ/ブログ管理**: 記事の作成、編集、削除、公開設定。
    *   **コンサート情報管理**: 公演情報の登録、編集。
    *   **会員限定コンテンツ管理**: 限定記事や動画の投稿。
*   **会員管理**:
    *   会員一覧表示、検索。
    *   会員ステータス管理（無料/有料/退会済）。
    *   決済状況の確認 (Stripe連携)。
